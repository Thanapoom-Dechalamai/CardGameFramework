Overview

Playable PrototypeOfflineHeads-upvs Bot

Texas Hold'em Offline เป็นระบบ heads-up poker ระหว่าง player และ bot ที่แยก game logic ออกจาก Unity presentation layer. แกนเกมอยู่ใน TexasHoldemRoundStateMachine ส่วน TexasHoldemOfflineGraphicDemo เป็น controller สำหรับ scene demo ที่ผูก UI, animation, audio และ visual feedback เข้ากับ state machine.

Current Scope: ระบบรองรับ playable loop ครบตั้งแต่เริ่ม hand, blinds/ante, betting rounds, bot action, all-in runout, showdown, winner result, chip animation, hand-rank preview และ auto next hand.

Core Capabilities

  • Round state machine สำหรับ PreFlop, Flop, Turn, River, Showdown และ HandComplete
  • Player actions: Fold, Check, Call, Bet, RaiseTo, AllIn
  • Live pot และ settled pot แยกกัน เพื่อรองรับ visual chip flow
  • Bot decision ใช้ equity simulation, pot odds, aggression, looseness และ bluff rate
  • Hand evaluator แยกเป็น domain layer ใช้ซ้ำกับ poker mode อื่นได้
  • Unity UI ใช้ serialized references จาก Inspector ไม่สร้าง UI control เองตอน runtime

Runtime Flow

Start Hand

  1. StartGame() หยุด coroutine เก่าและเริ่ม RunInteractiveHand()
  2. สร้าง TexasHoldemRoundStateMachine
  3. เรียก StartHand() พร้อม player ids, stack, dealer index และ bot ids
  4. State machine โพสต์ ante/blinds และ deal hole cards
  5. Controller animate การแจกไพ่บน Unity UI

Action

  1. Human กดปุ่ม หรือ bot เลือก action จาก SimpleTexasHoldemBot.ChooseAction(round)
  2. Controller เรียก round.ApplyAction(playerId, action)
  3. State machine validate turn/action และ resolve street
  4. Controller animate chips, board cards, text, timer และ rank

Showdown / End Hand

  1. ถ้า fold เหลือ active player คนเดียว จบด้วย fold win
  2. ถ้า all-in ถูก call แล้วไม่มี future betting ระบบ runout board จน river แล้ว showdown
  3. เปิด bot cards, แสดง rank, highlight best five cards
  4. Settled pot move to winner bet anchor, hold, แล้ว move เข้า profile
  5. Capture stack ต่อเนื่อง และ auto next hand

Decision Model

  • Monte Carlo-style equity estimation จาก hole cards + board cards
  • Pot odds เมื่อต้อง call
  • Personality randomization: aggression, looseness, bluff rate
  • Bet/raise sizing ตาม pot fraction
  • All-in range สำหรับ strong hand, short stack และบาง bluff spot

Code Examples

Example 1: Start and Play One Hand Without Unity UI

using Project.Application.Poker;
using Project.Domain.Cards;
using Project.Domain.Poker;

var comparer = new PokerHandComparer();
var bestEvaluator = new BestFiveCardPokerHandEvaluator(
    new PokerHandEvaluator(),
    comparer
);

var round = new TexasHoldemRoundStateMachine(
    new SystemRandomCardShuffler(),
    bestEvaluator,
    comparer,
    smallBlind: 400,
    bigBlind: 800,
    anteEnabled: true,
    anteAmount: 80
);

round.StartHand(
    playerIds: new[] { "Player", "Bot" },
    startingChipsByPlayer: new[] { 50000, 50000 },
    dealerIndex: 0,
    botPlayerIds: new[] { "Bot" }
);

while (!round.IsHandComplete)
{
    var current = round.CurrentPlayer;

    if (round.AmountToCall == 0)
        round.ApplyAction(current.PlayerId, TexasHoldemPlayerAction.Check());
    else
        round.ApplyAction(current.PlayerId, TexasHoldemPlayerAction.Call());
}

foreach (var result in round.RoundResult.Players)
{
    Console.WriteLine($"{result.PlayerId} winner: {result.IsWinner}");
}

Example 2: Human Action Buttons

public void OnMiddleActionPressed()
{
    if (!round.CanPlayerAct("Player"))
        return;

    if (round.AmountToCall == 0)
        round.ApplyAction("Player", TexasHoldemPlayerAction.Check());
    else
        round.ApplyAction("Player", TexasHoldemPlayerAction.Call());
}

public void OnRightActionPressed(int selectedRaiseTo)
{
    if (round.CurrentStreetBet == 0)
        round.ApplyAction("Player", TexasHoldemPlayerAction.Bet(selectedRaiseTo));
    else
        round.ApplyAction("Player", TexasHoldemPlayerAction.RaiseTo(selectedRaiseTo));
}

Example 3: Legal Bet Slider Range

int min;
int max;

var player = round.CurrentPlayer;

if (round.CurrentStreetBet == 0)
{
    min = round.GetMinimumLegalBet(player.PlayerId);
    max = player.Chips;
}
else
{
    min = round.GetMinimumLegalRaiseTo(player.PlayerId);
    max = player.StreetBet + player.Chips;
}

betSlider.minValue = min;
betSlider.maxValue = max;
